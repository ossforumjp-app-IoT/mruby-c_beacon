# mruby/c_beacon

このソースコードは、ビーコンアプリをmruby/cを使って動作させます。

sample_beacon_demo：mruby/cで記述したビーコンアプリのmruby/cコードとコンパイル後のcコードが格納されています。

src：mruby/cのソースコードが格納されています。
　　 一部のソースコードにアンアラインドデータ対策の修正を行っています。

※src内におけるmain.cは別途作成する必要があります。
